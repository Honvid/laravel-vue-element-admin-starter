# laravel-vue-element-admin-starter

this framework used the [laravel5.8](https://github.com/laravel/laravel), vue-element-admin, just a starter.

# 版本

- laravel 5.8
- vue-element-admin i18n 分支

# Getting Started

```
$ git clone https://github.com/Honvid/laravel-vue-element-admin-starter.git

$ cd laravel-vue-element-admin-starter

$ composer install

$ cp .env.example .env

$ php artisan key:generate

$ npm install

$ npm run dev

```
